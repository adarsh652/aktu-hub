# AKTU Hub UI Design System (v1)

## Vision

AKTU Hub should feel like a premium product rather than a typical
college website.

Design goals: - Clean - Fast - Minimal - Trustworthy - Student-focused -
Timeless (avoid trendy AI aesthetics)

Primary inspiration: - Apple --- whitespace, typography, hero sections -
Linear --- cards, spacing, subtle animations - Vercel --- monochrome
styling, buttons, layout - Notion --- educational simplicity - GitHub
--- structured tables and information hierarchy

------------------------------------------------------------------------

# Design Principles

1.  Content first
2.  Consistency everywhere
3.  Plenty of whitespace
4.  Minimal color palette
5.  Subtle animations only
6.  Accessibility and readability

------------------------------------------------------------------------

# Color Palette

Background: #FFFFFF

Secondary Background: #F8F9FA

Borders: #E5E7EB

Primary Text: #111827

Secondary Text: #6B7280

Primary Button: - Background: #111111 - Text: #FFFFFF

Success: - #22C55E

Warning: - #F59E0B

Error: - #EF4444

Avoid gradients and neon colors in the light theme.

------------------------------------------------------------------------

# Typography

Preferred fonts: - Inter - Geist

Hierarchy:

Hero Heading: 48--56px

Section Heading: 32--36px

Card Title: 20--24px

Body: 16px

Caption: 14px

Use regular and semibold weights. Avoid excessive bold text.

------------------------------------------------------------------------

# Layout

-   Maximum content width: \~1200px
-   Generous vertical spacing
-   Consistent margins
-   8-point spacing system

------------------------------------------------------------------------

# Border Radius

Use only: - 12px - 16px

Do not mix many corner styles.

------------------------------------------------------------------------

# Shadows

Use subtle shadows only.

Examples: - 0 1px 2px - 0 4px 12px

No heavy floating-card effects.

------------------------------------------------------------------------

# Navigation

Simple top navigation.

Logo

Resources

CGPA

About

Search

Responsive mobile menu.

------------------------------------------------------------------------

# Hero Section

Large welcoming heading.

Example:

Welcome to AKTU Hub

Everything an AKTU student needs --- Notes, PYQs, Syllabus, Results and
More.

Primary CTA: Explore Resources

Secondary CTA: Browse Semesters

Keep the hero clean without busy backgrounds.

------------------------------------------------------------------------

# Search

Large rounded search bar.

Placeholder:

Search notes, subjects, PYQs...

Instant filtering.

------------------------------------------------------------------------

# Cards

Subject cards should include: - Subject Name - Resource counts - Small
arrow

Style: - White background - Thin border - Rounded corners - Light hover
shadow - Slight lift on hover

------------------------------------------------------------------------

# Resource Pages

Display: - Notes - PYQs - Syllabus - Lab Files - Books (future)

Use clear section headings.

------------------------------------------------------------------------

# Tables

Minimal tables.

Thin borders.

Hover row.

No colorful headers.

------------------------------------------------------------------------

# Icons

Use Lucide outline icons consistently.

Examples: - Book - Folder - Graduation Cap - File - Search

------------------------------------------------------------------------

# Buttons

Primary: Black background White text

Secondary: White background Gray border

Hover: Slight darkening and elevation.

------------------------------------------------------------------------

# Empty States

Friendly messages.

Example:

📂

No resources available yet.

We're adding them soon.

------------------------------------------------------------------------

# Animations

Keep animations under \~250ms.

Allowed: - Fade - Hover lift - Smooth page transition - Smooth scrolling

Avoid: - Floating blobs - Excessive motion - Flashy transitions

------------------------------------------------------------------------

# Footer

Include: - AKTU Hub - Resources - About - Contact - GitHub - Made with
❤️ by Adarsh

------------------------------------------------------------------------

# Responsive Design

Desktop Tablet Mobile

Ensure: - Cards stack naturally - Search remains accessible - Navigation
collapses cleanly

------------------------------------------------------------------------

# Accessibility

-   High contrast text
-   Large tap targets
-   Keyboard-friendly navigation
-   Visible focus states

------------------------------------------------------------------------

# Future Dark Theme

Dark theme should reuse the same design language instead of becoming
colorful.

------------------------------------------------------------------------

# UI Philosophy

The user should immediately feel:

"This website is organized, trustworthy, and effortless to use."

The UI should never distract from the content. Every design decision
should help students find academic resources faster while maintaining a
premium, modern appearance.
